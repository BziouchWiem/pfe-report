\chapter*{CONCLUSION GÉNÉRALE}
\addcontentsline{toc}{chapter}{CONCLUSION GÉNÉRALE}
InvisiThreat a permis de formaliser un pipeline de détection combinant SAST et DAST,
de centraliser les résultats et d'offrir un tableau de bord exploitable pour les équipes
techniques. L'intégration de FastAPI, PostgreSQL et OWASP ZAP a assuré une base solide
pour l'automatisation, tandis que React et Socket.IO ont apporté une visualisation
réactive de l'état des scans.

Les perspectives immédiates incluent l'intégration CI/CD complète (scans déclenchés
à chaque merge), l'ajout d'un module SCA pour les dépendances tierces, et la mise en
place d'un moteur de corrélation afin de réduire les faux positifs. Une extension
vers le scanning d'images Docker et l'analyse de configuration cloud permettrait
d'élargir la couverture sécurité.
